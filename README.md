# Smart Glasses (name pending)
*CIT 460: Advanced Topics in Programming*
### Description:
The goal of the Smart Glasses project is to have an attachment for glasses that allows the user to detect and interact with text visible to the attached camera. The attachment will include a camera that will utilize the OpenCV library to detect visible text and an OCR engine to analyze it. With that text, the user can prompt the application through spoken action words after pressing the prompt sensor on the attachment. Given a prompt, the application will choose the appropriate API to use, such as LibreTranslate or Wolfram|Alpha. All API responses will either be played to the user through a connected Bluetooth audio device or available as a text response in the companion app. The secondary function of the attachment will be to take photos and videos that the user can save, view, and download through the mobile application.
